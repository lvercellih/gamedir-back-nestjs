# gamedir-back-nestjs

Backend para un directorio de videojuegos

## Tech Stack

- Node.js 18
- NestJS 10

## Iniciar el proyecto localmente
```bash
pnpm i
pnpm start:dev
```

## Integración
Pronto colocaré la especificación del api en **OpenApi** para que sea más fácil integrarlo con un frontend
